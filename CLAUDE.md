# drop-action

Zero-dependency, headless drag-and-drop primitives for React, exposing a
`createDropAction` factory whose public API mirrors a dnd-kit-style surface
(`Zone`, `Item`, `Active`, collision detection, modifiers) built on a custom
Pointer Events engine.

Read `CONTEXT.md` for the domain glossary and `docs/adr/` for the
architectural decisions before working in an area.
